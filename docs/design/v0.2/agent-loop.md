# Agent Loop

Agent Loop 指任务执行后的复盘和下一次任务的上下文复用，不属于 v0.1 MVP。

## 循环

```text
用户任务
-> LLM 生成脚本
-> Script Runtime 执行脚本
-> 保存完整运行轨迹和最终脚本
-> 复盘沉淀 Skill / Snapshot
-> 下一次任务把相关历史材料放入 Prompt
```

## 复盘输入

- 用户原始任务和对话。
- 最终脚本。
- 完整 RunTrace。
- 关键页面快照。
- 错误、下载、用户确认等执行事件。

## 输出

- Braiser Skill：网站能力知识。
- 可复用 Snapshot：关键页面状态和可验证结构。
- 修复建议：失败任务的定位和改写线索。

## 与 v0.1 的关系

v0.1 只需要保存足够调试的任务日志和最终脚本；完整 RunTrace 结构、自动复盘和历史检索留到 v0.2。
